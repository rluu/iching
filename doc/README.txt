Description:

Application to quickly make I Ching castings.

##############################################################################

Requirements:

Dependencies to build/run this project are:

  - Python 3.1
  - PyQt 4.7.5
  - Qt 4.6.3

##############################################################################

Steps to run the software:

1) Go to the project directory.

    [rluu@localhost iching]$ pwd
    /home/rluu/programming/iching

2) Change directory to the src directory.

    [rluu@localhost iching]$ cd src/

3) Run the executable.

    [rluu@localhost src]$ ./main.py
